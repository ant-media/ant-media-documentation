# License

- License Per Instance (Server) Based  
- No connection limit in/out

# Ingest Protocols

- [SRT](https://docs.antmedia.io/guides/publish-live-stream/srt/)  
- [WebRTC](https://docs.antmedia.io/guides/publish-live-stream/webrtc/) (1-1, 1-many, many-many, SFU, MCU, Data Channel)   
- [RTSP](https://docs.antmedia.io/category/ip-camera--stream-sources/#adding-an-ip-camera) (IP Cam, ONVIF Support, Remote PTZ Control)  
- [RTMP](https://antmedia.io/what-is-rtmp-server-how-to-set-up-a-free-rtmp-server/)  
- [Zixi](https://antmedia.io/marketplacetold/zixi-plugin-app/) 

# Streaming and Latency Values

- [WebRTC](https://docs.antmedia.io/guides/playing-live-stream/webrtc-playback/) with 0.5 sec latency  
- [LL-DASH](https://docs.antmedia.io/guides/playing-live-stream/dash-playing-cmaf/)/ [LL-HLS](https://antmedia.io/low-latency-hls-or-ll-hls/) (CMAF) with 3–5 sec latency  
- [HLS] (https://docs.antmedia.io/guides/playing-live-stream/hls-playing/) with 10–12 sec latency  
- RTMP for [Restream](https://docs.antmedia.io/guides/publish-live-stream/simulcasting/) with 3 sec latency  

# Scalability

- Vertical scaling based on CPU and memory  
- Horizontal scaling via [Kubernetes](https://docs.antmedia.io/category/kubernetes/) or [auto-scaling](https://docs.antmedia.io/category/clustering-and-scaling/) on cloud networks  

# ABR & Encoding

- [Adaptive](https://docs.antmedia.io/guides/adaptive-bitrate/adaptive-bitrate-streaming/) Bitrate on the fly  
- Software encoder  
- Hardware (GPU) encoder  
  - Compatible with [NVIDIA](https://docs.antmedia.io/guides/advanced-usage/using-nvidia-gpu/)  
  - Compatible with Intel Quick Sync  

# Codecs

- [H.264](https://docs.antmedia.io/guides/configuration-and-testing/video-codecs/)  
- [H.265](https://antmedia.io/the-first-hevc-h265-support-in-webrtc/) (HEVC)  
- [VP8](https://docs.antmedia.io/guides/configuration-and-testing/video-codecs/)  

# Recording & VoD

- [Recording](https://docs.antmedia.io/category/recording-live-streams/) formats: MP4, WebM, HLS  
- Recording to local HDD, [S3](https://docs.antmedia.io/category/s3-recording-and-integration/), or remote storage  
- [VoD](https://docs.antmedia.io/guides/recording-live-streams/playing-recorded-files/#playing-vod-streams-with-mp4) (Video on Demand)  
- Playlist support  
- Scheduler  

# Abstraction Layers

- Native [Android](https://docs.antmedia.io/category/sdk-integration/), [iOS](https://docs.antmedia.io/category/sdk-integration/), and [JavaScript](https://docs.antmedia.io/category/sdk-integration/) SDKs  
- [React](https://docs.antmedia.io/category/sdk-integration/) Native, [Flutter](https://docs.antmedia.io/category/sdk-integration/), [Unity](https://docs.antmedia.io/category/sdk-integration/) packages  
- [REST](https://docs.antmedia.io/category/rest-api-guide/) APIs to control all functions remotely  
- RTMP SDKs  
- [Plug-in](https://docs.antmedia.io/guides/developing-antmedia-server/introduction-plugin-structure/) architecture for direct integration with external applications  

# Security

- [Enable](https://docs.antmedia.io/category/stream-security/) / disable accepting undefined streams  
- [One-time](https://docs.antmedia.io/category/stream-security/) token control  
- [CORS](https://docs.antmedia.io/category/stream-security/) filter  
- [Hash-based](https://docs.antmedia.io/category/stream-security/) token  
- [Publisher](https://docs.antmedia.io/category/stream-security/) IP filter  
- [JWT](https://docs.antmedia.io/category/stream-security/) stream security filter  
- [Time-based](https://docs.antmedia.io/category/stream-security/) one-time password (TOTP)  
- [SSL](https://docs.antmedia.io/category/stream-security/)  
- [IP-based](https://docs.antmedia.io/category/stream-security/) whitelisting
