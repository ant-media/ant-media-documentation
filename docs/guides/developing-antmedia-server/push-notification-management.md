# Push Notification Management

Push notifications enable you to send announcements or video/audio call notifications to all users or specific groups. This page details how to send and configure these notifications.

## Prerequirements

- If you want to use Firebase Cloud Messaging, you need to create a Firebase account and you need to install the Firebase SDK into your client applications. You can follow [the official documentation](https://firebase.google.com/docs/cloud-messaging).
- If you want to use Apple Push Notifications, you need to create an Apple Developer  account and you need to follow [the official documentation](https://developer.apple.com/documentation/usernotifications).
